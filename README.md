# **[ 🚫 규칙 ]**

1 - 1. 스터디 Repository 를 Fork 합니다.</br>
1 - 2. 스터디 Repository 를 Clone 합니다.

2. Repository 에서 문제 작성 후 Push

3. 스터디 Repository 로 PR 요청
    - Assignees 본인으로 설정
    - Labels Subject 설정
        - Subject
            - ex) Computer Science



4. 코드 리뷰가 끝난 PR 방장이 Merge 진행
